Copy the code below and paste it into any page on your website — works with WordPress, Squarespace, Wix, Webflow, and any custom HTML site.

Step 2 — Add It to Your Site

Works with any website platform

Pick your platform below for specific instructions:

WP
WordPress: Open the page editor → add a Custom HTML block → paste the embed code → publish. Works with Elementor, Divi, and the block editor.
SQ
Squarespace: Edit your page → add a Code Block → paste the embed code → save.
WX
Wix: Click Add → Embed → Embed a Widget → paste the embed code into the HTML iframe panel.
WF
Webflow: Add an Embed element to your page → paste the embed code → publish.
HS
HubSpot / Other CMS: Use the HTML module or rich text editor in source mode → paste the embed code.
